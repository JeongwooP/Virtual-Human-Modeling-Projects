### Virtual-Human-Modeling
Computer Graphics Assignment: OpenGL primitive 모델링 함수들과 간단히 모델링 가능한 메쉬(ex. 직육면체, 실린더 등)를 사용하여 3D Virtual Human을 가능한 사실적으로 modeling 했습니다.
